# AI内容创作与发布分析系统

基于WorkBuddy能力构建的内容创作辅助工具，参赛作品 for AI黑客松 - 方向二【内容】

## 项目简介

AI内容创作系统是为个人创作者设计的智能辅助工具，旨在帮助创作者提升内容生产效率、优化多平台发布策略。

## 核心功能

### 1. 选题脑暴
- 输入领域/关键词，生成多个选题角度
- 每个选题包含：标题、角度、推荐平台、热度评分
- 提供选题理由和发布建议

### 2. 爆款拆解（规划中）
- 粘贴爆款文章链接或文本
- 自动分析：结构、标题技巧、情绪节点
- 提取可复用框架和优化建议

### 3. 多平台适配（规划中）
- 输入一篇文章，自动重写适配不同平台
- 支持：公众号、小红书、知乎
- 自动调整：语气、格式、长度

### 4. 标题AB测试（规划中）
- 输入文章主题，生成多个标题
- 按"点击吸引力"打分排序
- 提供标题优化建议

## 技术架构

- **后端**: Python + Flask
- **前端**: HTML5 + CSS3 + JavaScript + Bootstrap 5
- **AI能力**: 集成WorkBuddy API（当前为Demo模拟版本）

## WorkBuddy能力体现

1. **创意生成** - 选题脑暴功能
2. **结构分析** - 爆款拆解功能
3. **风格迁移** - 多平台适配功能
4. **多候选生成** - 标题AB测试功能

## 安装与运行

```bash
python app.py
```
